Transaction steps when using the escrow service:
1. The buyer pays a 4-8% ($3 minimum) service fee.
2. The seller designates the escrow agent as manager.
3. After 7 days, the seller assigns primary ownership rights to the escrow agent (7 days is the minimum amount of time required in order to assign a new primary owner in the control panel.)
4. The escrow agent verifies everything, removes the other managers, and notifies the buyer to pay the seller.
5. The buyer pays the seller.
6. After the seller's confirmation, the escrow agent assigns ownership rights to the buyer.
In order to guarantee maximum security during the transaction, all messages must be sent through the website using the chat system where the transaction is completed, so that in case of any issues, the arbitration service can verify everything.
